# towerdefense

## Installation

`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`    
`brew install git`     
`brew install python`

## Jouer

`python main.py`

## Configurer

Configuration :
1. des vagues dans `waves_config.py`,
2. des tours dans tower_config.py,
3. d'autres trucs dans constants.py.
